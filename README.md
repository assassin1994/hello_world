功能说明:
===========
一个简单的python程序，可以方便时间戳和普通日期的转换。
使用截图：
===========
<pre>
<code>
[root@barfoo3 ~]# tos 
1455788409000
[root@barfoo3 ~]# tos 20150101
1420041600000
[root@barfoo3 ~]# tos 20150101 20150101
[1420041600000 TO 1420127999999]
[root@barfoo3 ~]# tos 1455788409000
2016-02-18 17:40:09
</code></pre>
